RESIDUOLAB - Gestão de Resíduos Químicos

O projeto Resíduo Lab consiste no desenvolvimento de um sistema para gerenciamento de resíduos químicos, inicialmente voltado aos laboratórios de Química e Biologia do IFPR – Campus Paranavaí, com foco na organização, controle e descarte adequado desses materiais. O sistema é projetado com possibilidade de expansão futura, permitindo sua adaptação para outros contextos e instituições, encontrando-se atualmente em desenvolvimento.

JOGO DA MEMORIA

O projeto consiste no desenvolvimento de um mini jogo da memória utilizando JavaScript, HTML e CSS, com integração de APIs para obtenção dinâmica de dados e uso do Local Storage para armazenamento de informações, como pontuação e progresso do usuário. O objetivo é proporcionar uma experiência interativa, explorando conceitos de lógica de programação, manipulação do DOM e persistência de dados no navegador.
